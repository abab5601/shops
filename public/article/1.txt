@喝了那麽久的咖啡，今晨上了一課。@<br/>
<br/>
《咖啡豆》<br/>
作者：王冠雄<br/>
<br/>
黃金、黑金、綠金是世界三大生意，其中的綠金指的就是咖啡豆。咖啡樹生長於熱带或亞熱帶，土質疏鬆、陽光充足、並且降雨量豐富的地區，世界的三大主要產地是巴西、哥倫比亞與印尼。<br/>
<br/>
在三十多年前，我在蘇門答臘有一片林場，並進口黑檀、紫檀、花梨等高級板材，結識了一些蘇門答臘北部棉蘭地區的咖啡商，他們都很希望我幫助他們將生豆現貨（非期貨）賣到國際市場，因為現貨的價格約為期貨的兩、三倍，有很大的利潤空間。<br/>
<br/>
當時台灣進口咖啡豆是受管制的，因為哥斯達黎加政府還不出中華民國三千萬的外債，就協議以咖啡豆抵債，於是進口商就必須先向物資局申請，並將貨款繳交給物資局，再由物資局通知哥斯達黎加出貨。這種變相收取外債不但違反自由貿易的條例，並且哥斯達黎加藉機低價高賣品質較差的咖啡豆給台灣，經過進口商抗議之後，物資局才勉強同意以1 : 1 的方式開放進口，就是如果向哥斯達黎加進口一噸，就允許向其他國家進口一噸。<br/>
<br/>
因為不想被強迫進口哥斯達黎加的咖啡豆，我公司咖啡豆部門決定只進口研磨後的咖啡粉，而將咖啡生豆賣到國際市場。全盛時期，每個月經過我手轉賣的咖啡豆約2,000 – 3,000噸，約為台灣三年的總進口量。除了賣到歐洲、澳洲，甚至賣到北非卡薩布蘭卡與日本。其中，日本UCC咖啡每年共2100噸的購買量中，至少向我公司購買1200噸。原因無他，只因為我對品管的要求很嚴格，所以只有我公司敢接受信用狀中增列的一項條款：“如同已核准的樣品”。<br/>
<br/>
很多人迷惑於世界上很多熱帶國家都有種植，而且品牌多得無法細數，以為咖啡豆有很多不同的品種，其實，全世界的咖啡只有兩個品種，就是阿拉比卡與羅布斯塔，由於整個南美洲與非洲的咖啡樹都是阿拉伯人發現與引進，所以全都是阿拉比卡這同一品種，羅布斯塔僅生長於印尼，俗稱爪哇豆，佔印尼咖啡豆總產量的90%，其餘的10% 為阿拉比卡，集中種植於蘇門答臘北方棉蘭一帶。<br/>
<br/>
由於全世界的需求量大，咖啡豆也成為期貨買賣，現貨市場的價格就比較高。在國際市場上阿拉比卡 僅分為 DP與Mandarin green兩個等級，DP是指顆粒較大並經兩次篩檢，臭豆比例低於8%, Mandarin green （俗稱的曼特寧），臭豆比例約11 – 13%. 羅布斯塔（爪哇豆）分為AP與EK兩種等級，AP的顆粒較大，再細分為AP – Special 與 AP – 1，主要是賣給日本的。你如果去過日本街頭的咖啡店，門口那玻璃罐裡看到的那種顆粒很大的就是它，EK就是普通的爪哇豆，依臭豆的比例分為六級，雀巢咖啡買的就是這兩種臭豆比例較高的EK – 3、4，至於最差等級的EK – 5、6，就是你在台灣咖啡店所點的冰咖啡，又濃又甜，裡面亂七八糟添加了黃豆粉或玉米粉，或是太白粉之類等等的⋯⋯<br/>
<br/>
喝咖啡沒什麼太大的學問，最重要的咖啡豆的品質。最好喝的咖啡是以最簡單的方式沖泡的，就是將Arabica DP或Mandarin green烘培研磨的三號粉，以市面上最普通的美式濾泡咖啡機沖泡，這種濾泡方式以66 C 的水溫不會將過多的單寧酸與PH （酸鹼值）濾出。至於其他什麼88 C 或各種稀奇古怪的沖泡方式的都是在耍噱頭。最糟糕的是高溫高壓沖泡的義式濃咖啡Espresso，那是笨蛋才會喝的。 二次大戰時因物資缺乏，義大利人將幾顆咖啡都當寶貝，拼命的煮了再煮⋯⋯所有的酸鹼值，單寧酸，雜質都全在咖啡裡，然後，你就一口喝進了肚子裡⋯⋯<br/>
<br/>
烘培都是制式的，只要將機器設定好熱度與時間，想要咖啡豆有焦味，烘培的時間就長一點，哪有甚麼特殊技術可言？難不成是在烘培時跳段艷舞給咖啡豆看？<br/>
<br/>
咖啡店是一本萬利的好生意，有什會比以噸計價買進，然後一杯一杯的賣出的生意更好賺，以星巴克 為例，每杯平均美金7元，而且希望你拿了咖啡馬上就滾蛋，只要選的地點好，每開一家分店就如同挖到了一口油井。可是，我是從來不喝那種爛咖啡的。星巴克有各式各樣的口味，各種味道都是添加的，放根香草豆一起烘培，就有香草味；如果放塊奶油攪拌，就有牛油香，咖啡裡能變的花樣很多，加上一點鮮奶油就取名維也納咖啡，摩卡就是50%咖啡+50%熱巧克力……，我個人認為，星巴克設定的消費族群是雅痞族，就是那種看起來很聰明，他們也自以為很聰明，但其實很多方面是不聰明的那群人⋯⋯。<br/>
<br/>
以咖啡生豆現貨每噸約美金5,000元來計，每公斤約美金5元，生豆是以透氣的麻布袋包裝，每袋60公斤，保存期為三年。家庭用的小型烘培機與研磨機都非常便宜，就算你每天喝個5、6杯，這袋美金300元的咖啡豆三年也喝不完，而且你是享受最安全與品質最好的咖啡。更何況你永遠不知道外面買的咖啡，廠商在咖啡裡添加了什麼？不要忘了餿水油的例子，那些人什麼事都做得出來！如果你喝的是即溶咖啡，對你反而有害；如果選擇的是低咖啡因的咖啡，還不如不要喝。呵呵，再告訴你一個秘密，你知道有一種冰咖啡是什麼做的？那是拿黃豆炒焦後磨粉，加點糖精與咖啡精⋯⋯<br/>
<br/>
除了醫學研究證實喝咖啡的好處是：咖啡因可以促進大腦乙酰膽鹼的生成，那是大腦神經細胞最需要的傳導物質，減緩失智與巴金森氏症。至於坊間宣傳可以減肥、抗氧化、預防心肌梗塞、預防肝癌、大腸癌、降抵血醣、咖啡單寧酸可抑制新冠病毒⋯⋯，忽然之間，咖啡豆彷彿成了仙丹妙藥，其實，都是在胡說八道。<br/>
<br/>
除了抗衰老與抗癌之外，現在又有咖啡單寧酸可抗新冠病毒的說法，前一陣子說有這些功效的是綠茶，現在變成了咖啡，為了賣給你咖啡，這些人可以說盡所有的謊話。<br/>
<br/>
在非洲的肯亞國家公園內，曾經發生成群的長頸鹿離奇死亡的事件，在科學家及獸醫們追蹤調查後發現，死因是吃了刺槐的樹葉。那年適逢乾旱，水草不足，動物們開始大量的吃樹葉，而植物本身也有防禦的系統，刺槐在樹葉損失到一定的程度時，就會開始分泌大量的單寧酸，而當長頸鹿開始繼續食用刺槐頂端的嫩葉之後，後果就是中毒而死亡。<br/>
<br/>
咖啡或茶葉都含有單寧酸，尤其是即溶咖啡與隔夜濃茶中的含量更高。單寧酸對你最大的好處，就是能讓你心臟的冠狀動脈或腦血管內堆積的脂肪硬化，而心肌梗塞與腦中風是目前排名第二與第三的死因。<br/>
<br/>
凡事過猶不及，在你每天喝六杯以上的咖啡或是喝濃茶時，你最好也能同時想到，長頸鹿無論體積或重量可要比你大上很多！<br/>
<br/>
如果覺得自己烘培研磨太過於麻煩，在好市多有一種100% 阿拉比卡的咖啡粉，是我認為是目前市面上最好的，一罐2.5磅的阿拉比卡咖啡，美金15元的售價還算公道，以每天四杯來計，應該可喝一個月。超市也有賣各種口味的奶精，這就是我給大家的建議。
